# 📝 Feedback Form

A simple and clean feedback form built with **React.js** (frontend) and **MongoDB** (backend).  
Users can submit their feedback in real-time, and the data is securely stored in the database.

## 🚀 Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js + Express  
- **Database:** MongoDB
